Objective

Build a CodePen.io app that fulfills the user stories provided by FreeCodeCamp.

Original Challenge: https://www.freecodecamp.org/learn/coding-interview-prep/take-home-projects/build-a-wikipedia-viewer

User Stories

  1. I can search Wikipedia entries in a search box and see the resulting Wikipedia entries.
  2. I can click a button to see a random Wikipedia entry.
